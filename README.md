# BMI Health Calculator

This is a Full Stack BMI Health Calculator project developed using HTML, CSS, JavaScript, Node.js, Express, and MongoDB.

## Features:
- User Registration and Login (Authentication using JWT)
- BMI Calculation based on height and weight
- Health Status Display (Underweight, Normal, Overweight, Obese)
- Personalized Diet Suggestions
- MongoDB Database Integration
- Secure Password Hashing using bcrypt

## Technologies Used:
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Database: MongoDB

## How to Run:
1. Open server folder
2. Run `npm install`
3. Start server using `node index.js`
4. Open client files in browser
